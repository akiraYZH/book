# 码云GIT 操作
0. `git clone xxx`
## 本地上传代码到码云四步走
1. `git add .`
2. `git commit -m XXX`
3. `git pull`
4. `git push`


# 第一次操作
1. 打开terminal, `cd 路径` 转到相应目录下
2. `git clone gitee分享链接`
    >将.git文件夹粘贴到我们要上传文件夹目录下

# 设置免密码
1. 右击vs code中的带有git的文件夹
2. 打开terminal
    * `git config --global user.name 用户名`
    * `git config --global user.email 邮箱`
